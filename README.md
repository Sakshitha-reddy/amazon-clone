# Amazon Clone

A frontend Amazon clone created using HTML and CSS for learning and practice purposes.

This project focuses on recreating the layout and design of the Amazon homepage to improve frontend development skills and understand real-world website structure.

## Features
- Responsive navigation bar
- Search bar design
- Hero section/banner
- Product category boxes
- Footer section
- Clean UI layout using CSS

## Technologies Used
- HTML
- CSS

## Concepts Practiced
- HTML page structure
- CSS styling
- Flexbox layouts
- Positioning and alignment
- Margins, padding, and box model
- Responsive design basics

## Purpose
This project was built as a frontend practice project to strengthen HTML and CSS skills by cloning a popular real-world website interface.
